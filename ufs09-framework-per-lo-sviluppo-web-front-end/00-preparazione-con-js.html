Destrutturazione di un Array
...

Serve per accedere alle proprietà di un oggetto creando al tempo stesso variabile con il desiderato nella posizione 0, 1 ecc...
Estraggo il "resto" dell'oggetto (facoltativo).

const persone = ["Renzo", "Lucia", "Don Abbondio", "Don Rodrigo"];
const [persona1, persona2,...rest] = persone;

console.log(persona1)  //Renzo
console.log(persona2)  //Lucia
console.log(rest)      //["Don Abbondio", "Don Rodrigo"]

Un array in JavaScript non ha dati per forza omogenei, in quel caso si chiama tupla:

const tuple = [2, "Lucia"];
const [codice, nome] = tuple

console.log(codice)  //2
console.log(nome)    //Lucia

Oggetti in JS
...

Gli oggetti, chiamati anche array associativi, vengono dichiarati in questo modo:

const persona = {
	nome: "Renzo",    //chiavi
	cognome: "Lucia",
	eta: 20,
	altezza: 1.74
}

Per accedere alle proprietà di un oggetto, si usa la seguente sintassi:

const nomeRenzo = persona.nome

Destrutturazione di un Oggetto
...

La destrutturazione di un oggetto si fa in modo analogo agli array;

const {nome, cognome, ...rest} = persona.nome

Attenzione! Il nome delle variabili destrutturate deve essere lo stesso delle chiavi dell'oggetto.
Per rinominarle:

const {eta:etaRenzo} = rest

Spread Operator
...

Lo spread operator inserisce tutti i valori di un oggetto dentro un altro oggetto.

const anagrafica = {
	nome: "Renzo",
	cognome: "Tramaglino",
	eta: 20
}
const indirizzo = {
	via: "Dei Bravi",
	numeroCivico: 31
}

const persona = {...anagrafica, ...indirizzo}

Nullish coaliscence
...

const nome = persona?.nome ?? "Nessuna stringa"

Se persona e persona.nome esistono, assegna a nome persona.nome, altrimenti viene assegnato "Nessuna stringa".
