02 - TypeScript Tipi e Classi
tags:
  - typescript

Tipi di Dati
...

String
...

  • Doppie virgolette: Usate normalmente.
  • Singole virgolette: In alcuni casi preferite alle prime, si possono combinare quando serve.
  • Template Literals: Utili per evitare il concatenamento di stringhe usando il simbolo ${}.

Esempio di Codice
...

let string1 = "Hello";
let string2 = 'World';
let combinedString = `${string1} ${string2}`;

Number
...

  • Interi e decimali: Usati comunemente.
  • Operazioni aritmetiche: Esempi includono +, -, *, /.

Esempio di Codice
...

let number1 = 5;
let number2 = 10.5;
let sum = number1 + number2; // 15.5

Boolean
...

  • Valori: true o false.
  • Valori falsy: undefined, null, 0.
  • Conversione: Usare ! o !! per convertire in booleano.

Esempio di Codice
...

let isTrue = true;
let isFalse = false;
let isTruthy = !!1; // true

Strutture di Dati
...

Array
...

  • Caratteristiche: Sequenza di valori, solitamente omogenea.
  • Accesso: Tramite sintassi con parentesi quadre [index].
  • Funzioni utili: length, push(), pop().

Esempio di Codice
...

let array = [1, 2, 3];
array.push(4); // [1, 2, 3, 4]
let lastElement = array.pop(); // 4

Object
...

  • Caratteristiche: Inizializzato tramite parentesi graffe {}, coppie chiave-valore, può contenere altri oggetti.
  • Accesso: Tramite sintassi a punto . o con parentesi quadre ['key'].
  • Funzioni utili: Object.keys(), Object.entries().

Esempio di Codice
...

let person = {
  name: "John",
  age: 30
};
let keys = Object.keys(person); // ['name', 'age']
let entries = Object.entries(person); // [['name', 'John'], ['age', 30]]

Controllo di Tipo con TypeScript
...

Esempio 1: String
...

  • Aiutano a identificare cambi di tipo e suggeriscono proprietà delle classi e funzioni.

Esempio 2: Array
...

  • Inizializzazione di un array vuoto di tipo stringa in TypeScript, cosa non possibile in JavaScript.
  • Verifica durante la compilazione.

Esempio di Codice
...

let stringArray: string[] = [];

Esempio 3: Object
...

  • Definizione di tipi delle proprietà degli oggetti.
  • Creazione di un Custom Type Persona.

Esempio di Codice
...

type Persona = {
  name: string;
  age: number;
};

let persona1: Persona = { name: "John", age: 30 };
let persona2: Persona = { name: "Jane", age: 25 };

Esempio 4: Function
...

  • Definizione di funzioni in tipi personalizzati.
  • Problemi di duplicazione del codice.

Esempio di Codice
...

type FuncType = () => string;

let myFunc: FuncType = function() {
  return "Hello, World!";
};

Classi
...

Inizializzazione
...

  • Definizione delle proprietà e del costruttore.
  • Creazione di istanze tramite new.

Esempio di Codice
...

class Persona {
  name: string;
  age: number;

  constructor(name: string, age: number) {
    this.name = name;
    this.age = age;
  }

  getFullName(): string {
    return `${this.name} ${this.age}`;
  }
}

let persona1 = new Persona("John", 30);
console.log(persona1.getFullName()); // "John 30"

Suddivisione in Moduli ed Estensione delle Classi
...

Scopo
...

  • Raggruppare parti di codice comune.
  • Evitare classi troppo complesse.

Implementazione
...

  • Importazione del codice usando {}.
  • Estensione delle classi e uso dei decoratori.

Decoratori
...

  • Permettono di estendere una classe senza gerarchia.
  • Utilizzano il principio della programmazione funzionale.

Esempio di Codice
...

function MyDecorator(target: Function) {
  target.prototype.decorated = true;
}

@MyDecorator
class MyClass {
  constructor() {}
}

Uso di Angular
...

  • Component: Decoratore per gestire la vita della pagina, associa template e stili alla pagina.{}

Esempio di Codice in Angular
...

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title = 'my-app';
}
