Tests
tags:
  - vulnmanager
  - test

First of all, you have to enter the php docker container and cd into the vulnmanager directory:

docker exec -it docker-php-1 bash
cd vulnmanager

To run a test:

bin/phpunit --configuration=phpunit-new.xml.dist tests/Backend/<TestDirectory>/<Test>.php

If you want the test to stop after a failure, add the flag:

--stop-on-failure
